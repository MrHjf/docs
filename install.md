# 安装
[nmcc](https://www.npmjs.com/package/nmcc-cli)是一个脚手架工具，可用于快速搭建基于[nest.js](https://nestjs.com/)的node应用

## npm :
```npm install -g nmcc-cli```

```shell
# 全局安装 nmcc-cli
$ npm install -g nmcc-cli
# 创建一个基于nest.js新项目
$ nmcc new my-project
# 安装依赖
$ cd my-project
$ npm install
# 创建一个login文件夹
$ nmcc g login
```
